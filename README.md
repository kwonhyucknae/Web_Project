# Web_Project

사용자의 현재 위치를 위도와 경도로 받아 지도로 표시한 후


현재 위치와 가까운 매장을 마커로 표시해줬습니다.


매장 데이터는 공공데이터 포털 사이트에서 받아온 데이터를 오라클에 저장 후 Mybatis 를 이용해 쿼리문을 작성 후 데이터를 받아와 사용 했습니다.


그 외 게시판 기능에서는 게시판 목록과 글쓰기 글 수정, 댓글 기능을 구현 했습니다.
